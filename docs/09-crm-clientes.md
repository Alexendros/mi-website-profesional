## Pipeline de ventas

**Etapas:**

1. `Lead` — Contacto inicial, interés generico
2. `Cualificado` — Tiene presupuesto + timeline definidos
3. `Propuesta enviada` — Doc template de doc/08 enviado
4. `Negociación` — Ajustes de scope o precio
5. `Cerrado Ganado` — Contrato firmado + pago inicial recibido
6. `Cerrado Perdido` — Documentar razón
7. `Cliente Activo` — Subscripción activa en StageKit
